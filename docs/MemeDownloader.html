Meme Downloader

适用于 bilibili专栏链接提取脚本 的下载器哦

使用说明 📖

  1. 勾选需要下载的表情包 ✅，然后点击下方的【下载】按钮。
  2. 双击图片：切换预览图 🔄
  3. 单击图片：查看大图 🔍
  4. 退出大图模式：
    • 按 ESC 键。
    • 或点击右上角的叉号（×）。
    • 或再次单击任意处。

注意 ⚠️

  • 如果浏览器提示（通常在左上角）"此文件想要下载多个文件"，请点击允许。
  • 下载的表情文件将保存到当前 浏览器的默认下载文件夹 📁，图片文件将存放到 表情包名称.zip 的文件中。

如果你希望下载自己整理的TXT内容

请点击最下方的【加号】按钮，并粘贴或上传对应的TXT内容 📁。

再次提醒：本项目下载结果是 若干个压缩包，并不是直接返回图片
